# Redis

##### 主从复制特性

- 1.执行复制的从服务器连接到主服务器，接收主服务器发送的整个数据库的初始副本（copy）；
- 2.之后主服务器执行的写命令，都会被发送给所有连接着的从服务器去执行，实时更新从服务器。
- 3.客户端可以向任意一个服务器发送读的请求。

##### 数据结构

- string
  - 字符串，整数，浮点
  - 对整个字符串或字符串的一部分执行操作。
  - 对整数，浮点执行自增或自减操作。
- list
  - 一个链表，每个节点都是字符串
  - 从链表的两端推入或弹出元素。
  - 根据偏移量对链表进行修剪
  - 读取单个或多个元素
  - 根据值查找或移除元素
- set
  - 包含字符串的无序集合
  - 添加，获取，移除单个元素
  - 检查元素是否存在集合中
  - 计算交集，并集，差集
  - 从集合里面随机获取元素
- hash
  - 包含键值对的无序集合
  - 添加，获取，移除单个键值对
  - 获取所有键值对
- zset
  - 有序集合。
  - 字符串成员与浮点分值之间的有序映射，元素的书序由分值决定
  - 添加，获取，移除单个元素
  - 根据分值范围或成员获取元素
