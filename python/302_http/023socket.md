端口号：标记区分进程（区分电脑程序）。从0到65535

知名端口：0 -- 1023
80端口分配给HTTP服务
21端口分配给FTP服务

动态端口：1024 -- 65535
动态分配是指当一个系统进程或应用程序进程需要网络通信时，它向主机申请一个端口，主机从可用的端口号中分配一个供它使用。当这个进程关闭时，同时也就释放了所占用的端口号

# 查看端口
netstat －an
# 查看IP地址
ifconfig
--------------------------------------------
IP地址
用来在网络中标记一台电脑的一串数字，在本地局域网上是惟一的
由网络号Net_ID + 主机号Host_ID组成（Net_ID对应Network，二进制时说Net_ID，十进制时说Network）
# Net_ID有24bits
192.168.0.0/24
--------------------------------------------
IP分级
  0.xx.xx.xx ~ 127.xx.xx.xx # A类IP地址
128.xx.xx.xx ~ 191.xx.xx.xx # B
192.xx.xx.xx ~ 223.xx.xx.xx # C
224.xx.xx.xx ~ 239.xx.xx.xx # D
240.xx.xx.xx ~ 255.xx.xx.xx # E(保留，仅作实验和开发用)
--------------------------------------------
IP种类
IPv4分为2类：Public IP公有IP 和 Private IP私有IP

私有IP：只能内部网络使用，不能通过Internet传输
分为3级
10.0.0.0～10.255.255.255		# A
172.16.0.0～172.31.255.255	# B
192.168.0.0～192.168.255.255	# C
--------------------------------------------
IP地址127．0．0．1~127．255．255．255用于回路测试，
127.0.0.1可以代表本机IP地址，用http://127.0.0.1就可以测试本机中配置的Web服务器。
IP地址Host_ID全为0表示网络号Network IP，255表示广播地址Broadcast IP同一网段的端脑都能收到。这两个不能做主机IP使用
--------------------------------------------
网络掩码Netmask
和IP一起使用，按位与操作。可以得到网络号。可以判断IP地址类型
A类默认网络掩码255.0.0.0
B类默认网络掩码255.255.0.0
C类默认网络掩码255.255.255.0

子网划分：改变Net_ID的位数，Netmask也随着改变

创建socket
socket.socket(AddressFamily, Type)
Address Family：可以选择 AF_INET（用于 Internet 进程间通信） 或者 AF_UNIX（用于同一台机器进程间通信）,实际工作中常用AF_INET
Type：TCP协议SOCK_STREAM，UDP协议SOCK_DGRAM 

import socket
s1 = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s2 = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
